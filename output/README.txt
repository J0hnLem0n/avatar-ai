ПАПКА ДЛЯ РЕЗУЛЬТАТОВ
=======================

Здесь будут сохраняться созданные аватары:

ФОРМАТЫ ВЫВОДА:
- MP4 - видео с говорящим аватаром
- JPG - кадры из видео

НАСТРОЙКИ КАЧЕСТВА:
- Разрешение: 256x256 (по умолчанию)
- FPS: 25
- Кодек: H.264

СТРУКТУРА ИМЕН:
- result.mp4 - готовый аватар
- result_enhanced.mp4 - с улучшением качества

ПРИМЕЧАНИЯ:
- Время обработки зависит от длины аудио
- Качество зависит от входных файлов
- Поддерживает русскую речь
