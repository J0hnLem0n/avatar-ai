ПАПКА ДЛЯ ВХОДНЫХ ФАЙЛОВ
============================

Здесь размещайте файлы для создания аватара:

1. ФОТО (source_image) - изображение лица для аватара
   - Форматы: JPG, PNG
   - Рекомендуется: четкое фото лица в хорошем качестве
   - Размер: минимум 256x256 пикселей
   - Ракурс: фронтальный, лицо крупно

2. АУДИО (driven_audio) - речь для синхронизации
   - Форматы: MP3, WAV
   - Рекомендуется: четкая речь без шумов
   - Длительность: 10-60 секунд
   - Поддерживает русский язык

ПРИМЕРЫ ИМЕН ФАЙЛОВ:
- photo.jpg - фото лица для аватара
- speech.mp3 - аудио с речью
